# forum-website

A website for developing a forum across IIIT-NR

## build and run

```sh
python -m pip install pipenv
git clone https://github.com/DevWarriors-IIITNR/forum-website
cd forum-website
pipenv install
pipenv shell
python manage.py runserver
```
